# 🎵 Spotify Frontend Clone

A responsive **Spotify-inspired music player interface** built using **HTML, CSS, and JavaScript**.
This project recreates the modern Spotify UI design, focusing on layout, styling, and interactive components.

---

## 🚀 Features

* 🎧 Modern Spotify-inspired UI
* 📱 Responsive design
* ▶️ Play / Pause music controls
* 🎵 Music progress bar
* 📂 Song list display
* 🎨 Clean and professional layout
* ⚡ Smooth user interaction using JavaScript

---

## 🛠️ Built With

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout design
* **JavaScript (Vanilla JS)** – Player functionality and interactions
